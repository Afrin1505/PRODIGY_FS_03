# Product Image Assets

Place your product image files in this folder:

`client/src/assets/images/`

Example filenames used by the app:
- `apple.jpg`
- `banana.jpg`
- `orange.jpg`
- `milk.jpg`
- `bread.jpg`
- `rice.jpg`
- `sugar.jpg`
- `oil.jpg`
- `tomato.jpg`
- `potato.jpg`
- `onion.jpg`
- `eggs.jpg`
- `placeholder.jpg`

If you want to use your own images, keep the same file names or update `client/src/data/products.js` to point to your new files.

In React with Vite, product images are imported from this folder and used directly in product data.
